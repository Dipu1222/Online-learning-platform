A responsive **Online Learning Platform UI** built using **HTML, CSS, and JavaScript**.  
This project demonstrates the complete **front-end design and interaction flow** of an e-learning website.

> ⚠️ This is a frontend prototype. No backend or database is connected yet.

---

## 🌐 Repository

Hosted on **GitHub**: [https://github.com/Dipu1222/Online-learning-platform](https://dipu1222.github.io/Online-learning-platform/)

---

## 🚀 Live Demo (GitHub Pages)

You can deploy this project using GitHub Pages to see it live.

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- DOM Manipulation
- Responsive Design
- Form Validation

---

## ✨ Features

- 🏠 Hero landing section
- 📚 Dynamic course cards generated with JavaScript
- 📄 Course details modal popup
- 🛒 Checkout modal (UI demo)
- 🔐 Login / Signup / Reset Password modals (UI demo)
- 🧑‍🏫 Instructor application form with validation
- 📰 Dynamic blog section
- 🎨 Clean, modern, responsive UI with animations
